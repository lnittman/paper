# Paper

Design-to-code agent CLI. A purpose-built fork of [pi-mono](https://github.com/badlogic/pi-mono) for [Paper Desktop](https://paper.design).

See **[packages/coding-agent](packages/coding-agent)** for installation and usage.

## Development

```bash
npm install
npm run build
cd packages/coding-agent && npm link
paper
```
